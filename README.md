# Math engine!
**Written by Aman Singh**

Primarily for C++ practice

## Requirements
* OSX or Linux
* C++11
* CMake v3.17.2 + C++ compiler

## Setup

1. Clone
2. `cd` into the repository
3. Build
```bash
> mkdir build
> cd build/
> cmake ..
> make
```
4.  Run
```bash
# Once built, the CLI executable will be in the ./build directory
> ./mecli
```
